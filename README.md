# Fraud Academy Workstation

Fraud Academy Workstation is a fictional React + Vite + PWA training app for practicing fraud, chargeback, payroll, BEC, identity, credit risk, and banking investigation workflows.

## Build goals

- Brand-new repository build.
- React + Vite + PWA.
- Cozy neon investigator workstation theme.
- Unlimited generated fictional claims through a data-driven case generator.
- Claim-specific sidebar and investigation tools.
- Evidence-first flow that does not reveal the answer too early.
- Customer 360 with banking products, account status, balances, credit limits, and NSF/payment standing.
- Local progress saving with localStorage.

## Local commands

```bash
npm install
npm run dev
npm run build
```

## GitHub Pages

This repo includes a GitHub Actions deployment workflow. In GitHub Pages settings, select:

- Source: GitHub Actions

The app is configured for the `/legendary-broccoli/` GitHub Pages path.
